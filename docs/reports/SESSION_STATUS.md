### Session Class Methods Available:
- ✅ `start()` - Start session
- ✅ `isLoggedIn()` - Check if user is logged in
- ✅ `get($key, $default)` - Get session value
- ✅ `set($key, $value)` - Set session value
- ✅ `has($key)` - Check if key exists
- ✅ `remove($key)` - Remove session value
- ✅ `regenerate()` - Regenerate session ID
- ✅ `destroy()` - Destroy session
- ✅ `getUserId()` - Get logged-in user ID
- ✅ `getUserName()` - Get logged-in username
- ✅ `getUserRole()` - Get logged-in user role
- ✅ `setUser($userId, $username, $role)` - Set user session data
- ✅ `clearUser()` - Clear user session data
- ✅ `getUser()` - Get complete user data as array
- ✅ `setFlash($key, $message, $type)` - Set flash message
- ✅ `getFlash($key)` - Get and remove flash message
- ✅ `hasFlash($key)` - Check if flash message exists
