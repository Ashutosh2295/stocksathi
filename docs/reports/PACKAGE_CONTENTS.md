## 🛠️ Troubleshooting

### Issue: Installer shows blank page
**Solution:**
1. Check PHP version (must be 7.4+)
2. Verify Apache is running
3. Check error logs

### Issue: Cannot connect to database
**Solution:**
1. Ensure MySQL is running (green in XAMPP)
2. Use default credentials: root / (blank)
3. Verify database service is active

### Issue: Installation failed
**Solution:**
1. Check MySQL is running
2. Verify stocksathi_complete.sql exists
3. Try manual installation (see guide)

### Issue: Login page not found
**Solution:**
1. Verify folder structure is correct
2. Check .htaccess file exists
3. Ensure mod_rewrite is enabled
