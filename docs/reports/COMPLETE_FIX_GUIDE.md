## 🔧 TROUBLESHOOTING

### **If test-db.php shows red items:**

**Problem:** MySQL not running
**Solution:** Start MySQL in XAMPP Control Panel

**Problem:** Database doesn't exist
**Solution:** Create database "stocksathi" in phpMyAdmin

**Problem:** Users table doesn't exist
**Solution:** Import the SQL file or run the CREATE TABLE query

---

### **If registration shows error:**

The **simple-register.php** page will show you:
- ✓ Which step failed (in red)
- ✓ Exact error message
- ✓ File and line number if exception

**Common errors:**
- "Database connection failed" → MySQL not running
- "Table users doesn't exist" → Import SQL file
- "Email already exists" → Use different email

---

### **If login shows error:**

The **simple-login.php** page will show you:
- ✓ Login attempt status
- ✓ User role if successful
- ✓ Redirect URL
- ✓ Exact error if failed

**Common errors:**
- "Invalid credentials" → Wrong email/password
- "Database connection failed" → MySQL not running
