## ✨ HOW TO GET STARTED

### **Option 1: Fresh Setup (Recommended)**

1. **Go to Registration:**
   ```
   http://localhost/stocksathi/pages/register.php
   ```

2. **You'll see this message:**
   > 🎉 You're the first user! You'll get Super Admin access to set up your system.

3. **Register with your details:**
   - Name: Your Full Name
   - Email: your@email.com
   - Phone: Your phone number
   - Password: Choose a strong password

4. **Click "Create Account"**

5. **Wait 3 seconds** → Auto-redirect to login page

6. **Login** with your credentials

7. **🎊 You're now the Super Admin!** Full system access!

## 👥 ADDING STAFF MEMBERS

Once you're logged in as Super Admin:

1. Go to **Administration → Users**
2. Click **"Add User"**
3. Fill in details:
   - Name
   - Email
   - Password
   - **Role** (Choose from dropdown):
     - super_admin
     - admin
     - store_manager
     - sales_executive
     - accountant
4. Click **Save**
5. ✅ Staff member can now login!
