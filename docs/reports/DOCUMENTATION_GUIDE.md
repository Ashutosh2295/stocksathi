## 📋 How to Use These Files

### Option 1: Copy to Word/Google Docs

**Step 1:** Open the .md file in VS Code or any text editor
```
File → Open → docs\01_ACADEMIC_PROJECT_REPORT.md
```

**Step 2:** Select all content
```
Ctrl + A (Windows) or Cmd + A (Mac)
```

**Step 3:** Copy
```
Ctrl + C (Windows) or Cmd + C (Mac)
```

**Step 4:** Paste into Word/Google Docs
```
Open Word → New Document → Ctrl + V
```

**Step 5:** Format
- Font: Times New Roman 12pt
- Headings: Arial 14pt, Bold
- Line spacing: 1.5
- Margins: 1 inch all sides

---

### Option 2: Convert Markdown to PDF

**Using Typora (Recommended):**
1. Download Typora: https://typora.io/
2. Open .md file in Typora
3. File → Export → PDF
4. Mermaid diagrams will render automatically

**Using VS Code:**
1. Install "Markdown PDF" extension
2. Open .md file
3. Right-click → Markdown PDF: Export (pdf)

**Using Pandoc (Advanced):**
```powershell
pandoc 01_ACADEMIC_PROJECT_REPORT.md -o PROJECT_REPORT.pdf
```
